# 광고 패턴 라이브러리 (전환 기획 참조·축적)

ad-analyst가 전환 분석·시놉시스 기획 시 참조하는 검증된 패턴 모음. **새로 잘 먹힌 패턴(우리 성과·레퍼런스 광고)은 여기에 계속 추가**한다(Phase 4 피드백 루프의 자산).

## 1. 훅 유형 (첫 1~2초 — 숏폼 전환의 절반)
- **질문/공감형**: "이거 나만 그래?" / "Dry, rough knees? Same." — 타깃 페인 즉시 호명
- **충격 대비형**: 가장 심한 before를 먼저 보여줌(각질·고민 클로즈업)
- **숫자/시간형**: "7일 만에" / "한 번에" — 구체 수치(단, 효과 시점은 가설·근거 범위)
- **금기/반전형**: "비싸게 살 필요 없어요" / 통념 깨기
- **결과 선공개형**: 매끈한 after를 먼저 → "어떻게?"로 끌기
> 훅이 약하면 전환이 무너진다. 미지정 시 2~3개 훅을 만들어 비교.

## 2. 서사 프레임워크
- **PAS** (Problem-Agitate-Solve): 고민 제시→심화/공감→해결. `problem-solution` 템플릿.
- **AIDA** (Attention-Interest-Desire-Action): 주목→흥미→욕구→행동. 일반 광고 골격.
- **BAB** (Before-After-Bridge): 현재 고민→개선된 모습→제품이 다리. before/after형.
- **토킹헤드 리뷰** (Hook-Main-Closer): 진정성 있는 추천. `ugc-review`.

## 3. 소구 각도 (제품·타깃에 맞게 택1~2)
즉각 변화 / before-after 증거 / 신뢰·전문성(성분·인체시험) / 감성·자존감 / 가성비 / 희소성·트렌드

## 4. 숏폼 구조 원칙 (검증)
- **first-2s 훅** 필수 — 도입 인사/워밍업 금지, 첫 프레임부터 핵심
- **before/after 직접 비교**: 같은 부위·앵글이어야 설득력(좌우/각도 통일 — 가드레일)
- **UGC 진정성**: 하드컷, 과한 연출·스튜디오룩 지양, 자연 폰카메라 톤
- **CTA 명확**: 마지막에 행동 지시("Link in bio" 등)
- **클레임 안전**: 효능·외관 수준만, 의학/기능성 단정 금지(표시광고법)

## 5. 우리 사례에서 배운 것 (계속 축적)
- (2026-06) HAUA 무릎: before/after는 같은 다리·앵글이어야 "한 무릎이 변한 것"으로 읽힘. 컷마다 정면/좌향/우향 섞이면 "다른 무릎"으로 오인 → 앵글 통일 필수.
- (2026-06) 제형 정확도: 실제 제형컷 기반으로(흰 그레인 vs 진득 크림) — 안 맞으면 "딴 제품" 느낌.
- (추가 예정) 집행 성과(hook_rate·전환율)로 검증된 훅/앵글을 여기 기록.
