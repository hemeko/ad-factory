# 템플릿: problem-solution
서사(PAS): **Problem(고민 상황) → Agitate(공감·심화) → Solution(제품 등장·사용) → Result(개선·증거) → CTA.**
타깃의 페인포인트를 시각적으로 제시하고 제품이 해결하는 스토리텔링형 광고.

## 전환 가설
타깃이 "이거 내 얘기네"라고 즉각 공감(Problem/Agitate) → 해결책(Solution)과 결과(Result)로 구매 동기를 자극. 페인포인트가 뚜렷한 제품에 강함.

## 컷신 패턴(synopsis 생성 규칙)
- **problem**: 고민 상황·증상 제시(거울 앞 한숨, 거친 피부 클로즈업, 옷에 가리는 동작 등). start-only. 자막=고민 한 줄("이거 나만 그래?").
- **agitate**(선택): 고민 심화 또는 흔한 실패(평범한 제품으로 안 됨). 자막=공감 한 줄. (15초 이하 짧은 영상은 생략 가능)
- **solution**: 제품 등장·사용 동작(짜기·바르기·롤링 등), 기대 섞인 표정. start-only. 자막=제품/해결.
- **result**: 개선된 부위·표정·전후 대비. start-only 푸시인. 자막=결과.
- **endcard**: 제품컷 + 브랜드/제품명 + CTA.

## 모델/설정
- 이미지: nano_banana_2(9:16, 2k).
- 영상: **상황 액션·바디형이면 `kling3_0`(무음) + VO 오버레이**(PAS 내레이션) / **인물이 카메라에 말하면 `seedance_2_0`(립싱크)**. 디스커버리에서 결정.

## 일관성 절차
1. 마스터(인물 + 부위 + 구도) 확정 — problem 컷이 시작점.
2. solution/result는 마스터를 `--image` 레퍼런스로 변주(표정·제품 유무·상태만 변경).
3. **problem(문제)과 result(결과)는 같은 부위·좌우 측·앵글 고정**(가드레일) — 손동작 컷 포함 화면 방향 통일. 표정/상태로 변화를 준다.

## 일관성 vs 변화
- 고정: 인물·피부톤·부위·좌우·촬영 앵글.
- 변화: 표정(고민→만족), 제품 유무, 피부/부위 상태 개선.

## 가드레일(필수)
- guardrails/common.md 준수. 특히 **before/after(problem/result) 부위·좌우·앵글 일관성**, 클레임(제품 실제 효능·외관 수준만, 의학/기능성 단정 금지), NSFW 착장 명시.
- **언어 추측 금지**(VO/대사 언어는 디스커버리에서 확정).
- CTA 테일 또는 엔드카드 CTA 필수.
