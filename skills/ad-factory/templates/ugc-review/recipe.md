# 템플릿: ugc-review
서사 3단: **Hook(시선 사로잡기) → Main(특징·사용감 리뷰) → Closer(추천·소감)**.
크리에이터가 카메라에 직접 말하는 토킹헤드형 광고/리뷰.

## 핵심 특징
- **모놀로그 + 립싱크**: ad-analyst가 작성한 모놀로그 대본을 seedance_2_0으로 립싱크 영상 생성.
- **Step 자막 없음**: 단계 자막 대신 크리에이터의 행동·대사 중심. 자막은 훅·CTA만.
- **하드컷**: 진정성 있는 UGC 편집. 디졸브/페이드 금지.

## 보드 구성
- **16:9 한 장에 9:16 슬롯 3개** (tutorial의 21:9×4와 구별):
  - Slot 1 — Hook: Tight(클로즈업) 또는 Mid(바스트샷). 미드액션 훅 순간.
  - Slot 2 — Main: Mid(바스트) 또는 Wide(상반신). 제품 사용·보여주기.
  - Slot 3 — Closer: Mid 또는 Tight. 자신감 있는 마무리 표정.
  - 3슬롯 거리 대역 교차 필수(Tight·Mid·Wide 중 2~3가지 혼용).
- 보드 오버레이 금지(Zero Overlay Chrome): 텍스트·화살표·플레이스홀더 금지, 순수 실사만.

## 컷신 패턴(synopsis 생성 규칙)
- **hook**: 미드액션 훅 순간(정적 미소 금지). 모놀로그 hook 대사 립싱크. 자막=훅 문구(선택).
- **main**: 제품 들고/사용하며 설명. 모놀로그 main 대사 립싱크. 자막 없음 또는 키워드.
- **closer**: 마무리 소감·추천. 모놀로그 closer 대사 립싱크. 자막 없음.
- **cta_tail**: 마지막 0.5~1초, 한 손으로 아래 가리키기 + "Link in bio". 자막=CTA 문구.
  (cta_tail은 closer 클립 말미에 결합하거나 별도 1s 클립으로 추가)

## 모델/설정
- 이미지 보드: nano_banana_2(16:9 wide, 2k) 또는 gpt_image_2(레이아웃 안정).
- 영상(립싱크): **seedance_2_0** (립싱크·오디오·다중 레퍼런스 지원). kling3_0은 토킹헤드 비적합(무음·립싱크 없음).

## 인물 일관성 (특정 인물=Soul ID / 가상 인물=페르소나 마스터+보드)
- **특정 인물을 재현**해야 하면 → Soul ID 경로(ad-persona, `--soul-id <ref_id>`).
- **가상의 크리에이터**(특정인 불필요)면 → 페르소나 마스터 1장(`frames/persona.png`)을 확정한 뒤 **3슬롯 보드**로 일관성 확보. Soul 학습 크레딧이 들지 않고, 이번 실전(흑인 여성 크리에이터)에서 검증된 경로. persona.png를 보드 생성의 `--image` 레퍼런스로 준다.
- 어느 경로든 보드 3슬롯은 동일 인물로 고정. seedance_2_0 생성 시 보드 슬롯 이미지 + (Soul ID 또는 persona 레퍼런스) + 모놀로그 오디오를 다중 레퍼런스로 주입.

## 가드레일 (필수)
guardrails/common.md 전체 준수. 이 템플릿에서 특히:
- Beauty Floor 앵커 구문 + Modesty Triplet 의상 구문 필수.
- 촬영 구도: 고정 카메라(핸즈프리) 정면 기본 / 셀카는 옵션(arm extended 강제 안 함), `cool neutral daylight`. 촬영 장비 프레임 노출 금지.
- 팔레트 일치: 의상색 ↔ 배경 인테리어색.
- 그립-표정 물리: 제품 무게감에 따른 손 자세 + 표정.
- CTA 테일 필수: 마지막 클립 말미 한 손 아래 가리키기.
- 게이트: 페르소나(Soul ID 또는 마스터) 서브게이트 → 보드 3슬롯 확인 게이트1 → 전체 클립 게이트2 → 최종 게이트3.
